# Quartz Runtime

Quartz Runtime is the future standalone loader for Quartz native mods.

Current status:
- Reads enabled Quartz mods from runtime/enabled-manifest.json
- Stages enabled packages into runtime/staged
- Writes logs into runtime/logs
- Validates package staging

Not implemented yet:
- Native Geometry Dash loading
- Hooking
- Quartz native mod API
- In-game runtime
