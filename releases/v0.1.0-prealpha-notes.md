# Quartz Launcher v0.1.0 Pre-Alpha

This is an early pre-alpha tester build of Quartz Launcher.

## Downloads

- Windows x64 portable ZIP
- Linux x64 portable ZIP
- macOS x64 unsigned portable ZIP

## Important notes

- This is a pre-alpha build for testing only.
- macOS build is unsigned.
- Quartz-native runtime testing exists, but actual Geometry Dash native loading/injection is not finished yet.
- Geode compatibility packages are supported while Quartz-native support continues to grow.

## Current status

- .quartz package format
- package validation
- install/uninstall
- enable/disable
- runtime staging
- runtime test command
- Cloudflare website
